# A-PERSONAL-CHEF---Made-with-LANGCHAIN-GEMINI-and-DUCKDUCKGO
A short Langchain project that takes a list of ingredients from the user and returns some recipes. It uses the Google Gemini model to create the agent and DuckDuckGoSearchRun from Langchain community tools as a web search tool. 
Make sure to use your own Gemini API key in the .env file as the existing key doesn't work.
